数据流:

数据在Input时,从SMG架构图底层流向高层;
理解的过程其实就是数据的处理过程;

性能优化:

1,性能塔:(见数据流)底层流向高层的同时,数据量大量减少;只保留了Law,Value,Logic,MappingKnowledge等表的数据;
2,"唯一性的性能优化":避免了理解分析的重复;避免了同样数据的重复;在理解与分析决策性能上优化效果良好;
基于:类比、归纳与统计的理解系统:

1,理解是数据的关联;
2,建立关联的效率效果远大于深度学习的效率效果;(达到看两眼西红柿即认得的效果)
3,只有基于理解的NLP才有可能达到人类同等水平;
先天空白:

1,先天数据库无数据;
2,先天无分词,无任何数据;
基于能力开发:

1,不依赖算法;只保留取属性和基础对比;两个算法;
2,基于视觉,听觉,发声,识别字符串等基础能力的开发;后天用基础能力构建无限通用复杂的能力;
